# clj-http-server
A Simple HTTP Server in Clojure

## Running The Server
lein run -p 5000 -d path/to/cobspec/public

## Running CobSpec
make sure CobSpec is located in the same dir as the server.
cd to cobSpec
run java -jar fitnesse.jar -p 9090
navigate to localhost:9090

## Testing
lein speclj
